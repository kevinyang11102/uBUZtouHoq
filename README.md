# 学校宿舍管理系统 python1041

## 项目概述

学校宿舍管理系统是基于 Python 的 Django 框架开发的一个适用于校园宿舍管理的应用平台。它旨在提高宿舍管理的效率，规范学生住宿行为，为校园宿舍管理提供便捷的服务。

## 技术栈

- 后端框架：Python Django
- 数据库：PostgreSQL
- 前端框架：Bootstrap

## 功能模块

1. 学生信息管理
   - 学生基本信息录入
   - 住宿分配与调整
   - 学生信息查询与统计
2. 宿舍楼管理
   - 宿舍楼信息维护
   - 宿舍房间状态管理
3. 维修服务
   - 维修申请提交
   - 维修进度跟踪
4. 安全管理
   - 出入记录管理
   - 紧急事件处理

## 系统角色

- 管理员：负责整个系统的维护与监控
- 宿舍管理员：负责宿舍楼的具体管理工作
- 学生：使用系统进行住宿相关操作

## 运行环境

- 操作系统：Linux 或 macOS（推荐），Windows（支持）
- Python 版本：3.6 或以上
- 数据库：PostgreSQL 10.0 或以上

## 部署步骤

1. 安装 Python 3.x
2. 安装 PostgreSQL 数据库
3. 克隆项目代码
   ```bash
   git clone [repository-url]
   ```
4. 安装依赖
   ```bash
   pip install -r requirements.txt
   ```
5. 配置数据库
6. 迁移数据库
   ```bash
   python manage.py makemigrations
   python manage.py migrate
   ```
7. 运行开发服务器
   ```bash
   python manage.py runserver
   ```
